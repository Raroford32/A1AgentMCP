Tool Execution Endpoints

POST /api/tools/execute/source-code-fetcher
Body Parameters: 
{
  "contractAddress": "0x...",
  "chainId": 1
}
Example: 
curl -X POST http://localhost:5000/api/tools/execute/source-code-fetcher \
  -H "Content-Type: application/json" \
  -d '{"contractAddress":"0x1234...","chainId":1}'

SSE (Server-Sent Events) Endpoints

Connect to these endpoints to receive real-time updates from tool executions:

GET /sse/tools/source-code-fetcher
GET /sse/tools/constructor-parameters
GET /sse/tools/state-reader
GET /sse/tools/code-sanitizer
GET /sse/tools/concrete-execution
GET /sse/tools/revenue-normalizer

SSE Connection Example

const eventSource = new EventSource('http://localhost:5000/sse/tools/source-code-fetcher');

eventSource.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Tool update:', data);
};

eventSource.onerror = (error) => {
  console.error('SSE error:', error);
};
